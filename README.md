# レシピスクショ管理アプリ

レシピ投稿のスクショ画像を保存し、タグで整理・検索できるスマホ向けWebアプリです。  
OCRは使わず、画像とメタ情報（名前・タグ・メモ）で管理します。

## 機能

- スクショ画像アップロード＆保存（LocalStorage）
- レシピ名、タグ複数、メモ、お気に入り登録
- キーワード検索（レシピ名・タグ・メモ）
- タグ絞り込み
- カード一覧表示
- 画像クリックで拡大表示
- 編集・削除
- 新しい順表示

## 技術

- HTML / CSS / JavaScriptのみ
- GitHub Pagesで公開可能
- データ保存: LocalStorage

## 使い方

1. 画像を選択して、レシピ名・タグを入力
2. 必要に応じてメモ、お気に入りを設定
3. 保存ボタンで登録
4. 検索欄にキーワードを入力して検索
5. タグで絞り込み
6. 画像タップで拡大表示
7. 編集/削除ボタンで管理

## 注意

- LocalStorage容量には上限があります。
- データは同一ブラウザ・同一端末内のみ保持されます。
