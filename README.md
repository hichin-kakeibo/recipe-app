# レシピスクショ管理アプリ

レシピ投稿のスクショ画像を保存し、タグで整理・検索できるスマホ向けWebアプリです。  
OCRは使わず、画像とメタ情報（名前・タグ・メモ）で管理します。

## 機能

- スクショ画像アップロード＆保存（LocalStorage）
- レシピ名、タグ複数、メモ、お気に入り登録
- キーワード検索（レシピ名・タグ・メモ）
- タグ絞り込み（複数タグのAND条件）
- カード一覧表示
- 画像クリックで拡大表示
- 編集・削除
- 新しい順表示

## 技術

- HTML / CSS / JavaScriptのみ
- GitHub Pagesで公開可能な静的構成
- データ保存: LocalStorage（画像はData URL形式）

## 使い方

1. 画像を選択して、レシピ名・タグ（カンマ区切り）を入力
2. 必要に応じてメモ、お気に入りを設定
3. 保存ボタンで登録
4. 検索欄にキーワードを入力して検索
5. 「タグで絞り込み」からタグを選択して分類検索
6. 画像タップで拡大表示
7. 各カードの編集/削除ボタンで管理

## GitHub Pages公開

1. GitHubへpush
2. リポジトリの `Settings > Pages`
3. Sourceを `Deploy from a branch`、Branchを `main`（または公開対象ブランチ）/`root` に設定

## 注意

- LocalStorage容量には上限があります。画像が多い場合は容量超過の可能性があります。
- データは同一ブラウザ・同一端末内のみ保持されます。
