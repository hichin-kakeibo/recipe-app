# レシピスクショ収納

スマホのスクショフォルダに溜まったレシピ画像を、**タグで仮想フォルダ管理**するためのアプリです。  
OCRは使わず、画像そのものを保存して、あとから検索しやすくすることを最優先にしています。

## 主な機能

- レシピ画像アップロード（画像自体をLocalStorageに保存）
- レシピ名入力
- タグ複数登録（カンマ区切り）
- メモ入力
- お気に入り登録
- キーワード検索（対象: レシピ名 / タグ / メモ）
- タグ絞り込み検索（複数タグAND検索）
- 画像カード一覧表示
- 画像タップで拡大表示（モーダル）
- 編集・削除
- 新しい順表示
- よく使うタグのワンタップ追加

## 使い方

1. 画像を選び、レシピ名・タグ（必須）・メモを入力
2. 必要なら「お気に入りにする」をON
3. 「保存する」で登録
4. 検索欄にキーワード入力で絞り込み
5. 「タグ一覧」からタグを複数選ぶと仮想フォルダのように絞り込み可能
   - 例: `豚肉` と `冷凍` を両方選ぶと、両タグ付きのみ表示
6. 一覧画像をタップすると拡大表示
7. カードの「編集」「削除」で管理

## 技術構成

- `index.html`: 画面構造
- `style.css`: ベージュ系モバイルUI
- `script.js`: 保存・検索・描画・編集ロジック
- データ保存: `localStorage`（画像は Data URL として保存）

## デザイン設定

- 背景: `#F5F0E8`
- メインカラー: `#C4956A`
- 文字: `#3D3D3D`
- やさしいベージュ系、スマホで見やすいカードUI

## GitHub Pages 公開

1. GitHubにpush
2. `Settings > Pages`
3. Sourceを `Deploy from a branch`、Branchを `main` / `root` に設定
4. 公開URLでアクセス

## 注意

- LocalStorageの容量には上限があります。高解像度画像を大量保存すると上限に達する場合があります。
- 同一ブラウザ・同一端末内でのみデータが保持されます。
