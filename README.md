# レシピメモ（MVP）

スマホ（特にiPhone）で使いやすい、手入力型のレシピ保存アプリです。  
HTML / CSS / JavaScript だけで動作し、データは LocalStorage に保存されます。

## 機能

- レシピの登録（レシピ名 / 画像URL / 材料 / 作り方 / メモ / タグ）
- レシピ一覧のカード表示
- キーワード検索（名前・材料・作り方・メモ・タグ対象）
- タグ検索（検索欄にタグ文字を入力）
- お気に入り登録（☆/★）
- レシピの編集・削除
- LocalStorage 永続化

## ファイル構成

- `index.html` : 画面構造
- `style.css` : ベージュ系のシンプルでかわいいスタイル
- `script.js` : CRUD / 検索 / お気に入り / LocalStorage 制御

## 使い方

1. `index.html` をブラウザで開く
2. フォームにレシピ情報を入力して **保存する** をタップ
3. 一覧カードから以下を操作可能
   - **編集**: フォームに内容を読み込み、更新保存
   - **削除**: 確認ダイアログ後に削除
   - **☆/★**: お気に入り切り替え
4. 上部の検索欄でキーワードやタグを入力して絞り込み
5. 「お気に入りのみ」にチェックするとお気に入りだけ表示

## GitHub Pages で公開する

1. このリポジトリを GitHub に push
2. GitHub リポジトリの **Settings > Pages** を開く
3. **Build and deployment** で以下を設定
   - Source: `Deploy from a branch`
   - Branch: `main`（または公開したいブランチ） / `/ (root)`
4. 数分待つと公開URLが発行される

## 補足

- データはブラウザの LocalStorage に保存されます。
- 同じ端末・同じブラウザ内でのみ保持されます。
- OCR機能は未実装（MVPのため手入力のみ）。
